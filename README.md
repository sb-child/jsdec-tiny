# jsdec-tiny
a tiny version for https://github.com/liulihaocai/JSDec/ | jsdec 缩小版
## 如何使用
### 编译
<需要事先安装`golang`>  
+ windows用户:
  ```bash
    # 运行 build.cmd
  ```
+ linux用户:
  ```bash
    ./build.sh
  ```
### 运行
+ windows用户:
  ```cmd
    cd build
    main.exe
    (输入要解密的js的base64或使用管道, 以换行符作为结束符)
  ```
+ linux用户:
  ```bash
    cd build
    ./main
    (输入要解密的js的base64或使用管道, 以换行符作为结束符)
  ```
